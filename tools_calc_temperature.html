calc_temperature

Temperature calculation module for clumped isotope analysis.

This module provides functions to calculate clumped isotope values (D47, D48, D63, D64, D65) as a function of temperature using polynomial equations from published calibrations (Hill et al., 2014; Fiebig et al. 2021, 2024).
